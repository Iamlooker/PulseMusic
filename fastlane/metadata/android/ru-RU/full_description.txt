Приложение для проигрывания вашей оффлайн коллекции музыки с поддержкой плейлистов, тем оформления и многим другим.

Необходимые разрешения:
1. READ_EXTERNAL_STORAGE: Для получения музыки из внешнего хранилища
2. FOREGROUND_SERVICE: Для проигрывания музыки в фоновом режиме
3. WRITE_EXTERNAL_STORAGE: Для просмотра обложек альбомов (необходимо для Android 9 и ниже)

Опциональные разрешения:
3. BLUETOOTH: Для проверки состояния Bluetooth-устройств, и автоматического запуска воспроизведения
4. RECEIVE_BOOT_COMPLETED: Для автоматического запуска приложения после перезагрузки телефона (позволяет использовать такие функции, как авто-воспроизведение при подключении к Bluetooth-устройствам)
